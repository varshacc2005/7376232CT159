HOW IT MAINTAINS TOP 10 EFFICIENTLY:

When new notifications arrive continuously:

  1. Use MIN-HEAP data structure (size = 10)
  2. For each new notification:
     ├─ Calculate priority score: O(1)
     ├─ If heap < 10: Insert O(log 10) ≈ O(1)
     └─ If new_score > min_score: Replace min O(log 10) ≈ O(1)
  3. Result: O(m log k) = O(m) for m new notifications

Why Placement won't get pushed out:
  └─ Placement weight (3.0) × 100 = 300 base points (minimum)
  └─ Even oldest Placement (300) > newest Event (max 150)
  └─ Type weights mathematically prevent displacement
